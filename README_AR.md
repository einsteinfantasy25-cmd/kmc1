# KMC B27 Grade Calculator Bot - Final Release

هذه النسخة النهائية تتضمن:

- حساب معدل المرحلة الأولى حسب 15 مادة و 36 credits.
- الحساب بالتقديرات أو بالدرجات الرقمية.
- زر **ضعيف** بدل راسب في التقديرات.
- تقرير HTML احترافي بصفحة A4 واحدة قدر الإمكان.
- التقرير لا يعرض أي Telegram ID أو username للطالب.
- لوحة أدمن خاصة تظهر فقط للأدمن.
- قائمة المستخدمين والإحصائيات وتقارير آخر 24 ساعة.
- نظام حظر احترافي:
  - فحص يدوي من لوحة الأدمن.
  - كشف تلقائي عند حظر/إيقاف البوت عبر my_chat_member.
  - حفظ المحظورين بقائمة خاصة.
  - منع المستخدم المحظور من استخدام البوت عند العودة.
  - حظر يدوي عبر Telegram ID أو @username.
  - رفع حظر يدوي عبر Telegram ID أو @username.

## الملفات المطلوبة للرفع على GitHub

ارفع واستبدل هذه الملفات:

- bot.py
- requirements.txt
- Procfile
- logo.png
- README_AR.md
- nixpacks.toml

بعدها اضغط Commit changes، ثم من Railway اضغط Redeploy.

## متغيرات Railway المطلوبة

- BOT_TOKEN = توكن البوت من BotFather
- ADMIN_IDS = Telegram ID الخاص بالأدمن

للحصول على ID الأدمن، اكتب للبوت:

/myid

ثم ضع الرقم في Railway داخل ADMIN_IDS.

## أوامر الأدمن المخفية

الأوامر /admin و /myid و /rename تعمل عند كتابتها يدويًا، لكنها لا تظهر في قائمة الأوامر العامة للمستخدمين.

## ملاحظة مهمة عن كشف الحذف

Telegram يستطيع إخبار البوت إذا المستخدم عمل Block/Stop للبوت. أما مجرد مسح المحادثة بدون حظر فليس له إشعار رسمي من Telegram.
