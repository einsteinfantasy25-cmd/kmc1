# KMC B27 Grade Calculator Bot

نسخة PDF احترافية بصفحة A4 واحدة.

## آخر تعديل
- تقرير الطالب PDF بصفحة واحدة قدر الإمكان.
- تغيير زر التقدير من "راسب" إلى "ضعيف".
- إبقاء "راسب" كمدخل مقبول إذا كتبه الطالب يدويًا، لكن لا يظهر كزر.
- الحفاظ على لوحة الأدمن والإحصائيات وملفات آخر 24 ساعة.

## التشغيل على Railway
ارفع الملفات إلى GitHub ثم اعمل Redeploy.
المتغيرات المطلوبة:
- BOT_TOKEN
- ADMIN_IDS
