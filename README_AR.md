## إعداد BotFather المقترح

### About / Short Description
بوت حساب معدل المرحلة الأولى لطلاب كلية طب الكندي - Batch 27.

### Description
بوت KMC B27 Grade Calculator يساعد طلاب المرحلة الأولى على حساب معدلهم حسب الكردتات، إما بالتقديرات أو بالدرجات الرقمية، ويرسل تقريرًا مرتبًا باسم الطالب يوضح معدل المرحلة وتأثيرها في التراكمي النهائي.

### Bot Picture / Animated media
من BotFather ادخل إلى:
/mybots → اختر البوت → Edit Bot → Edit Botpic
ثم ارفع الصورة أو الوسائط التي يقبلها BotFather لحساب البوت.
