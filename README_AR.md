# KMC B27 Grade Calculator Bot

نسخة احترافية نهائية:

- تقرير PDF بتصميم A4 كامل.
- اسم الطالب يظهر داخل التقرير فقط، بدون Telegram username أو ID أو أي تفاصيل حساب.
- دعم أقوى للأسماء العربية داخل التقرير عبر توليد التقرير كصورة A4 عالية الدقة ثم حفظها PDF.
- جدول المواد يمتد بعرض الصفحة ويستفيد من كامل الورقة.
- وقت التقرير حسب توقيت العراق.
- لوحة أدمن خاصة بمدير البوت فقط.
- حفظ تقارير آخر 24 ساعة وإرسالها للأدمن عند الطلب.
