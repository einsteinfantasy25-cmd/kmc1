# KMC B27 Grade Calculator Bot - نسخة الأدمن والأسماء العربية

## التحديثات
- يقبل اسم الطالب بالعربي أو الإنكليزي.
- اسم ملف PDF يكون باسم الطالب الذي يدخله.
- لوحة الأدمن تظهر فقط لمدير البوت بعد تفعيل ADMIN_IDS.
- لوحة الأدمن تعرض: إحصائيات، آخر 50 مستخدم، وتصدير CSV لكل المستخدمين.

## الملفات التي ترفعها على GitHub
ارفع هذه الملفات واستبدل القديمة:
- bot.py
- requirements.txt
- Procfile
- logo.png

## تفعيل لوحة الأدمن
1. افتح البوت في Telegram.
2. اكتب الأمر:
   /myid
3. البوت سيرسل لك رقم حسابك في Telegram.
4. افتح Railway > مشروع البوت > Variables.
5. أضف Variable جديدة:
   Name: ADMIN_IDS
   Value: رقم حسابك الذي أرسله البوت

مثال:
ADMIN_IDS = 123456789

إذا تريد أكثر من أدمن:
ADMIN_IDS = 123456789,987654321

بعد إضافة ADMIN_IDS اضغط Deploy/Redeploy.

## استخدام لوحة الأدمن
بعد التفعيل افتح البوت واكتب:
/admin

أو اضغط زر:
🛠 لوحة الأدمن

ستظهر لك أزرار:
- 📊 إحصائيات
- 👥 قائمة المستخدمين
- 📤 تصدير CSV

## ملاحظة
عدد المستخدمين يبدأ من بعد رفع هذه النسخة؛ المستخدمون الذين استخدموا النسخة القديمة قد لا يظهرون إلا إذا استخدموا البوت مرة أخرى.
