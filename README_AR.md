# KMC B27 Grade Calculator Bot

## آخر تعديل احترافي

هذه النسخة تحافظ على أجزاء البوت السابقة: الحساب، لوحة الكيبورد، لوحة الأدمن، قائمة المستخدمين، وفحص الحالة، مع تعديل جذري لتقرير PDF.

### تغييرات التقرير
- التقرير صار صفحتين A4 بدل حشر كل التفاصيل في صفحة واحدة.
- الصفحة الأولى: اسم الطالب، التاريخ حسب العراق، المرحلة، الكلية، والنتيجة النهائية فقط.
- الصفحة الثانية: جدول المواد الكامل بعرض الصفحة.
- لا تظهر أي معلومات Telegram داخل تقرير الطالب.
- دعم أقوى للاسم العربي داخل PDF عبر خطوط DejaVu/Matplotlib ونظام رسم يدعم RTL.
- لا يوجد تقسيم أو عبارة “من أصل 5”؛ تظهر مساهمة المرحلة كنسبة من المعدل النهائي.
